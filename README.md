# Garden VR -  Joc pentru reabilitare

**Garden VR** este un joc VR in care utilizatorul planteaza seminte, uda plantele si culege flori pentru a forma un buchet.

Scopul jocului este de a incuraja miscarea controlata a mainilor prin activitati simple si naturale de gradinarit. Proiectul este gandit pentru a fi usor de configurat si adaptat in functie de nevoile utilizatorului.

Jocul este structurat pe etape clare (Planting → Watering → Harvesting) si foloseste interactiuni bazate pe hand tracking.

---

##  Functionalitati principale

- Configurarea numarului de ghivece prin parametri JSON  
- Ajustarea distantei de lucru si a inaltimii mesei  
- Alegerea tipului de floare sau selectie random  
- Etape progresive:  
  - **Planting** – plantarea semintelor  
  - **Watering** – udarea plantelor  
  - **Harvesting** – culegerea florilor  
- Formarea unui buchet in palma utilizatorului  
- Suport pentru mana stanga, dreapta sau ambele  
- Reset intre runde de joc   

Proiectul este integrat cu un sistem de sesiuni si permite personalizarea comportamentului jocului prin parametri externi.
